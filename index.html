About me

As a technical writer, I apply the core writing concepts: minding the audience, keeping the content task-oriented, providing step-by-step instructions, providing useful visuals, and using clear and simple language. I created the style guide, so it is easier to keep the documentation portal unified.

I am passionate about UI and UX. I took a few courses to get a deeper understanding of design and managed to put this knowledge into practice by designing new tools for the product we are building. I believe that proper microcopy and user-friendly interfaces improve products, and I hope to work more in design in the future.

Recently I started to study the basics of web development: HTML, CSS, and JavaScript. Currently I am learning CSS grid, flexbox, and other fundamental layout concepts. In fact, this page is built in order to test these features in action.
